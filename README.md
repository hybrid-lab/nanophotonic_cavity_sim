# nanophotonic_cavity

general_GDS_patterns is just a file used to generate various GDS patterns, like alignment markers or some simple geometries. 

There are two main pipelines: one for the simulation of the bandstructure in mbp in the bandstructure_class.py (the bandstructure_tutorial explains how to use the class) and one for the simulation of the cavity in tidy3d (the cavity_tutorial explains how to use the various classes to simulate in tidy3d various properties of the cavity).
